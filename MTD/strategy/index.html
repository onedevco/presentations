OAuth 2.0 with Next.js

Seamless authentication using API routes

Next.js API Route Structure

// app/api/auth/login/route.ts export async function GET() { const authUrl = `https://api.service.hmrc.gov.uk/oauth/authorize ?response_type=code &client_id=${process.env.HMRC_CLIENT_ID} &scope=read:self-assessment write:self-assessment &redirect_uri=http://localhost:3000/api/auth/callback`; return NextResponse.redirect(authUrl); } // app/api/auth/callback/route.ts export async function GET(request: Request) { const code = new URL(request.url).searchParams.get('code'); const tokens = await exchangeCodeForTokens(code); // Store tokens in encrypted cookie or database return NextResponse.redirect('/dashboard'); }

Why Next.js Makes This Easier

  • No separate server - Callback handled by /api/auth/callback
  • Environment variables - .env.local keeps secrets safe
  • Server-side tokens - Never exposed to client JavaScript
  • Automatic refresh - Middleware checks token expiry
  • Session management - Store tokens in httpOnly cookies

Redirect URI Configuration

Register http://localhost:3000/api/auth/callback as your redirect URI in the HMRC Developer Hub for both sandbox and production.

Risks & Mitigations

Potential challenges and how to address them

High Risk

Production Approval Delays

HMRC approval can take 2+ weeks and may require iterations

Mitigation: Start early, follow checklist precisely, test fraud headers thoroughly

High Risk

API Changes Before April 2026

HMRC may update endpoints or requirements

Mitigation: Monitor HMRC changelog on GitHub, build flexible API client

Medium Risk

Fraud Header Compliance

Headers must be accurate or HMRC will reject production access

Mitigation: Use established npm packages, test with HMRC's header validation tools

Medium Risk

Data Mapping Errors

Incorrect category mapping could affect tax calculations

Mitigation: Use HMRC's CSV mapping files, validate against sandbox

Low Risk

OAuth Token Expiry

Tokens expire and need refreshing

Mitigation: Implement automatic refresh with retry logic

Low Risk

Spreadsheet Format Changes

Your data format may evolve over time

Mitigation: Build configurable column mappings
